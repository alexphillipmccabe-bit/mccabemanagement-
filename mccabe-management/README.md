# McCabe Management — React + Tailwind (Vite)

## Quick start
1. Install: `npm install`
2. Run locally: `npm run dev`
3. Build: `npm run build` (outputs to `dist/`)

## Deploy (Vercel, GitHub route)
- Push this folder to a GitHub repo, then import it in Vercel → Deploy.

## Deploy (Netlify drag-and-drop)
- Run `npm run build`. Drag the `dist/` folder onto https://app.netlify.com/drop
